# Moto Events Bot 🏍🇪🇸

Telegram-бот мото-афиши Испании. Раз в день:

1. **Обходит топ-30 постоянных источников** из [sources.json](sources.json) — календари концентраций, трек-дни, Circuit Ricardo Tormo, агенды Аликанте/Валенсии, мото-прессу, пресс-релизы DGT. Claude извлекает события и новости из дайджеста.
2. **Дополнительно ищет** через встроенный веб-поиск Claude то, чего в постоянных источниках нет (локальные quedadas, афиши клубов).
3. **Пишет** посты на русском — не дословный перевод, а простой читабельный пересказ: заголовок, 2–3 коротких абзаца, что это значит для мотоциклиста, ссылка на источник.
4. **Прикладывает картинку**: афишу/фото из статьи (og:image со страницы источника). Если картинки нет — пост текстом с превью ссылки.
5. **Постит** в канал и пересылает в группу по графику (только в рабочие часы Испании, `WORK_START`–`WORK_END`, с паузой между постами):
   - анонс — примерно за месяц до события (или сразу, если нашлось позже);
   - напоминание — за неделю;
   - пост «Сегодня!» — в день события.

События хранятся в `data/events.json` — дубликаты не постятся повторно.

## Что нужно перед запуском

1. **Бот**: у @BotFather → `/newbot` → получить токен.
2. **Канал**: создать канал, добавить бота **администратором** (право публиковать).
3. **Группа**: два варианта:
   - привязать группу к каналу как «Обсуждение» — Telegram сам пересылает посты, `GROUP_ID` оставить пустым;
   - или добавить бота в группу и указать её ID в `GROUP_ID` — бот перешлёт сам.
4. **Ключ Anthropic**: [console.anthropic.com](https://console.anthropic.com) → API Keys.

## Установка на Windows VM

```
git clone <repo> moto-events-bot
cd moto-events-bot
make-env.bat          ← вводишь токены
npm install
npm run test-post     ← тестовый пост в канал (проверка прав)
npm run run-now       ← полный цикл прямо сейчас (поиск + публикация)
install-autostart.bat ← фоновая работа через PM2
```

Обновление после `git push`: `update.bat`.

## Команды

| Команда | Что делает |
|---|---|
| `npm start` | Демон: постит каждый день в `POST_HOUR`:00 по Испании |
| `npm run run-now` | Разовый полный цикл (сбор + публикация) |
| `npm run test-post` | Тестовое сообщение в канал |
| `npm run sources` | Проверить доступность всех источников (без Claude/Telegram) |

## Настройки (.env)

| Переменная | Описание |
|---|---|
| `TELEGRAM_BOT_TOKEN` | токен бота |
| `CHANNEL_ID` | `@имяканала` или `-100…` |
| `GROUP_ID` | группа для пересылки (пусто = не пересылать) |
| `ANTHROPIC_API_KEY` | ключ Claude |
| `CLAUDE_MODEL` | `claude-sonnet-5` (по умолчанию — достаточно для извлечения/перевода/постов) или `claude-opus-4-8` (максимум качества) |
| `POST_HOUR` | час запуска по Europe/Madrid (по умолч. 10) |
| `WORK_START` / `WORK_END` | окно постинга по Испании (по умолч. 9–21); вне окна посты откладываются |
| `POST_INTERVAL_SEC` | пауза между постами в одном запуске (по умолч. 45 сек) |

## Стоимость

Два запроса к Claude Sonnet 5 в день: дайджест источников (~25–30k токенов входа) + веб-поиск (до 6 поисков, $10 за 1000). Итого ориентировочно **$0.10–0.30 в день**.

## Как подкрутить

- **Добавить/убрать источник**: отредактируй `sources.json` (имя + URL, RSS или обычная страница — бот сам разберётся), проверь `npm run sources`.
- Лимиты постов за день: `MAX_ANNOUNCES_PER_RUN`, `MAX_NEWS_PER_RUN` в `src/index.js`.
- За сколько дней анонсировать: `ANNOUNCE_WINDOW_DAYS` (по умолч. 35).
- Тон и правила постов: промпты в `src/collect.js`.
- Удалить событие / поправить дату: отредактировать `data/events.json` и перезапустить.
